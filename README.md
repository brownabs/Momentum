# Momentum

Momentum is a web application for software developers to stay motivated and keep track of their side projects. Users are able to create projects, add comments to see how their progression has gone, and mark projects as completed. Sometimes seeing other's progression is a good way to stay motivated. Users can follow and unfollow friends, and see what projects they are working on. Fully implemented CRUD, and created this project using C# with ASP.NET and Entity and Identity Frameworks.
